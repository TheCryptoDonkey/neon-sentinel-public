# Neon Sentinel

A browser arcade rescue shooter built with Vite, TypeScript, Canvas2D, WebGL,
Nostr score signing, and value-for-value support.

## Development

```sh
npm ci
npm run dev
```

## Checks

```sh
npm test
npm run build
```

## Deployment

Production deployment credentials, signer secrets, origin hosts, Cloudflare
details, and Caddy operator settings belong outside the public repo.

## Art And Audio

The shipped game art, music, and sound effects live under `public/`. Local
source/reference material is not required to play or build the game.
